# Orbit - Social CRM

Map your relationships. Prune the noise.

A relationship management app to track, nurture, and visualize your personal and professional connections.

## Features

- **Contact Management**: Store details about people in your life
- **Relationship Health Score**: Track relationship based on rec vitalityency, frequency, and energy
- **Interaction Logging**: Record how you connect with people
- **Relationship Map**: Visual overview of your social circles
- **Insights**: Understand patterns in your relationships

## Tech Stack

- React Native (Expo)
- Zustand for state management
- Local-first with optional sync

## Getting Started

```bash
npm install
npm start
```

## Concepts

### Relationship Health Score
Calculated based on:
- **Recency** (30%): Days since last interaction
- **Frequency** (30%): How often you interact
- **Energy** (40%): How interactions make you feel

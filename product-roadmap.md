# Product Roadmap: Orbit (Social CRM)

## 1. Product Identity

- **Name:** Orbit
- **Tagline:** Map your relationships. Prune the noise.
- **Core Function:** Social CRM for mapping and managing personal relationships
- **Strategy:** Users who trust Brier with their anxiety data will trust Orbit to help manage relationships

## 2. Core Features

### The Relationship Map
- Visual graph of all personal/professional relationships
- Color-coded by relationship health/type
- Connection strength indicators

### Relationship Scoring
- Track energy exchange (gives/takes)
- Historical interaction data
- "Puppet Master" detection (toxic relationships)

### Interaction Logging
- Log interactions with contacts
- Track conversation topics
- Set reminders for follow-ups

### Pruning Tools
- Suggestions for relationship maintenance
- "Graduation" tracking for healing relationships
- Healthy boundaries setting

## 3. Technical Architecture

| Layer | Technology | Notes |
|-------|------------|-------|
| Frontend | React Native (Expo) | Reuse from Brier |
| Local DB | SQLite | Offline-first |
| Sync | Tame Cloud | E2E Encrypted |
| Backend | Node.js + PostgreSQL | Supabase |

## 4. Integration with Brier

- Brier users can migrate to Orbit
- Share encrypted relationship data
- Pattern correlation (anxiety triggers from specific people)

## 5. Monetization

- Part of Tame Cloud subscription
- Free tier: 50 contacts
- Pro tier: Unlimited contacts + insights
